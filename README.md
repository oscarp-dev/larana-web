# Larana Web – Tienda Online

Proyecto web full stack desarrollado como tienda online para una joyería. Incluye sistema de usuarios, gestión de productos y conexión a base de datos MySQL.

## Tecnologías
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Funciones principales
- Inicio de sesión de usuarios
- Gestión de productos
- Navegación estructurada
- Interfaz web visual

## Instalación
1. Clonar el repositorio
2. Configurar la base de datos MySQL
3. Ejecutar el proyecto en servidor local
