# 💎 Larana Web – Tienda Online

Aplicación web full stack desarrollada como tienda online para una joyería.  
Incluye sistema completo de autenticación de usuarios, gestión de productos, carrito de compra y conexión a base de datos MariaDB.

---

## 🚀 Tecnologías utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Composer

---

## ✨ Funcionalidades principales

- 🔐 Registro e inicio de sesión de usuarios
- 👤 Gestión de perfil de usuario
- 🛍 Gestión de productos (crear, editar, eliminar)
- 🛒 Sistema de carrito de compra
- 📦 Procesamiento de pedidos
- 📜 Historial de compras
- 📧 Envío de correos
- 🗂 Navegación estructurada
- 🖥 Interfaz web organizada y funcional

---

## 🛠 Instalación y ejecución local

### 1️⃣ Clonar el repositorio

git clone https://github.com/oscarp-dev/larana-web.git

### 2️⃣ Acceder al directorio del proyecto

cd larana-web

### 3️⃣ Instalar dependencias con Composer

composer install

### 4️⃣ Configurar base de datos

- Crear una base de datos en MariaDB.
- Importar el archivo SQL.
- Configurar las credenciales en:

tienda_login_php/db_connect.php

Ejemplo:

$host = "localhost";
$user = "root";
$password = "";
$dbname = "tienda";

### 5️⃣ Ejecutar en servidor local

Colocar el proyecto dentro de:

C:\xampp\htdocs\

Y acceder desde el navegador:

http://localhost/J_S25_Tienda_Online
